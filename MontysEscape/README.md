# Monty's Escape - Unity 2D Platformer

## 📖 Описание проекта
**Monty's Escape** — это 2D платформер в ретро-стиле (8-16 bit) с процедурно генерируемыми уровнями, где игрок управляет героем Монти, который пытается сбежать из опасной лаборатории.

### Основные характеристики:
- **Жанр**: 2D платформер / аркада
- **Платформа**: iOS / Android
- **Стиль**: Пиксель-арт, ретро эстетика 80-90х
- **Длительность уровня**: 30-60 секунд
- **Сложность**: Хардкорная, но честная

---

## 🎮 Core Gameplay Loop
1. Игрок заходит в уровень
2. Изучает паттерны (враги, ловушки)
3. Пытается пройти
4. Умирает → мгновенный рестарт (<1 сек)
5. Доходит до выхода
6. Получает награду
7. Открывает следующий уровень

## 🔧 Установка и настройка

### Требования:
- Unity 2021.3 LTS или новее
- 2D URP (Universal Render Pipeline)

### Шаги установки:

1. **Создайте новый 2D проект в Unity**
   - Откройте Unity Hub
   - Создайте новый 2D проект

2. **Импортируйте скрипты**
   - Скопируйте все файлы из `Assets/Scripts/` в ваш проект

3. **Настройте слои (Layers)**
   - Создайте слой "Ground" для земли и платформ
   - Создайте слой "Player" для игрока

4. **Настройте теги (Tags)**
   - Добавьте тег "Player" для объекта игрока

5. **Создайте материалы и префабы**
   - Создайте префабы для игрока, врагов, платформ и других объектов

## 🎯 Настройка объектов

### Игрок (Player)
1. Создайте 2D объект (квадрат или спрайт)
2. Добавьте компоненты:
   - `Rigidbody2D` (Freeze Rotation = true)
   - `BoxCollider2D`
   - `PlayerController` (скрипт)
3. Настройте параметры:
   - Move Speed: 8
   - Jump Force: 12
   - Coyote Time: 0.15
   - Jump Buffer: 0.1
4. Создайте пустой дочерний объект "GroundCheck" для проверки земли
5. Добавьте тег "Player"

### Враг (Enemy)
1. Создайте 2D объект
2. Добавьте компоненты:
   - `Rigidbody2D` (Freeze Rotation = true)
   - `BoxCollider2D`
   - `Enemy` (скрипт)
3. Создайте два пустых объекта для точек патрулирования (PointA, PointB)
4. Настройте скорость и параметры

### Шипы (Spike)
1. Создайте 2D объект (треугольник или спрайт шипа)
2. Добавьте компонент `Spike` (скрипт)
3. Настройте как триггер

### Движущаяся платформа (MovingPlatform)
1. Создайте 2D объект (платформа)
2. Добавьте компоненты:
   - `BoxCollider2D`
   - `MovingPlatform` (скрипт)
3. Создайте две точки для движения (PointA, PointB)

### Ключ (Key)
1. Создайте 2D объект
2. Добавьте компонент `Key` (скрипт)
3. Настройте как триггер
4. Укажите ID ключа

### Дверь (Door)
1. Создайте 2D объект
2. Добавьте компонент `Door` (скрипт)
3. Настройте как триггер
4. Укажите требуемый ключ и следующую сцену

1. File → Build Settings → Android
2. Switch Platform
3. Настройте Player Settings:
   - Company Name
   - Product Name: Monty's Escape

2. Switch Platform
3. Настройте Player Settings

## 🎮 Уровни (World 1 - "Escape Facility")

### Уровень 1 — "Первый шаг"
- Обучение движению
- Плоская поверхность без опасностей

### Уровень 2 — "Простые прыжки"
- Добавляются небольшие пропасти

### Уровень 3 — "Высота"
- Вертикальные платформы

### Уровень 4 — "Первая ошибка = смерть"
- Добавляются шипы

### Уровень 5 — "Комбинация"
- Прыжки + шипы

### Уровень 6-7 — "Движение"
- Движущиеся платформы

### Уровень 8-9 — "Первый враг"
- Враги с патрулированием

### Уровень 10 — "Микс"
- Комбинация всех механик

### Уровень 11-12 — "Ключ"
- Ключи и двери

### Уровень 13-20 — Продвинутые уровни
- Вертикальные челленджи
- Ложные пути
- Таймеры
- Финальный тест
