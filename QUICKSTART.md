# Monty's Escape - Quick Start Guide

## 🚀 Быстрый старт за 5 минут

### Шаг 1: Создание проекта Unity
```
1. Откройте Unity Hub
2. Нажмите "New Project"
3. Выберите шаблон "2D Core"
4. Назовите проект "MontysEscape"
5. Нажмите "Create Project"
```

### Шаг 2: Импорт скриптов
```
1. В проекте Unity создайте папку Assets/Scripts
2. Скопируйте все .cs файлы из этой папки в Assets/Scripts
```

### Шаг 3: Настройка игрока
```
1. Создайте 2D объект: Right-click в Hierarchy → 2D Object → Sprites → Square
2. Переименуйте в "Player"
3. Добавьте компоненты:
   - Add Component → Rigidbody2D
     * Freeze Rotation Z = true
   - Add Component → BoxCollider2D
   - Add Component → PlayerController (скрипт)
4. Создайте дочерний объект:
   * Right-click на Player → Create Empty
   * Назовите "GroundCheck"
   * Позиция: (0, -0.5, 0)
5. В PlayerController скрипте:
   * Ground Check Point → перетащите объект GroundCheck
   * Ground Layer → выберите слой земли
6. Добавьте тег "Player" объекту Player
```

### Шаг 4: Создание уровня
```
1. Создайте землю:
   * 2D Object → Sprites → Square
   * Назовите "Ground"
   * Растяните до нужного размера
   * Добавьте BoxCollider2D
   * Создайте новый слой "Ground" и назначьте на объект

2. Создайте платформу:
   * Повторите шаг 1 для платформ
```

### Шаг 5: Добавление врага
```
1. Создайте 2D объект для врага
2. Добавьте компоненты:
   - Rigidbody2D (Freeze Rotation Z = true)
   - BoxCollider2D
   - Enemy (скрипт)
3. Создайте два пустых объекта для точек патрулирования:
   * PointA и PointB
   * Разместите их на расстоянии
4. В скрипте Enemy:
   * Point A → перетащите PointA
   * Point B → перетащите PointB
```

### Шаг 6: Добавление шипов
```
1. Создайте спрайт шипов (треугольник или импорт)
2. Добавьте компонент Spike (скрипт)
3. Убедитесь что Collider настроен как Trigger
```

### Шаг 7: Настройка камеры
```
1. Найдите Main Camera
2. Добавьте компонент CameraFollow (скрипт)
3. Target → перетащите объект Player
4. Настройте Offset: (0, 2, -10)
```

### Шаг 8: Создание GameManager
```
1. Создайте пустой объект: Create Empty
2. Назовите "GameManager"
3. Добавьте компонент GameManager (скрипт)
4. Настройте ссылки:
   * Player → перетащите Player
   * Spawn Point → создайте пустой объект в точке спавна
```

### Шаг 9: Тестирование
```
1. Нажмите Play в Unity
2. Управление:
   * Стрелки или WASD — движение
   * Пробел — прыжок
3. Игрок должен двигаться и прыгать!
```

## 🎨 Добавление графики

### Где взять спрайты:
- [Kenney.nl](https://kenney.nl/) - бесплатные ассеты
- [OpenGameArt.org](https://opengameart.org/)
- [Itch.io](https://itch.io/game-assets)

### Импорт спрайтов:
```
1. Перетащите PNG файлы в папку Assets/Sprites
2. Выберите спрайт в Inspector
3. Настройте:
   * Filter Mode: Point (no filter) для пиксель-арта
   * Pixels Per Unit: 16 или 32
   * Compression: None
```

## 📱 Мобильное управление

Для добавления экранных кнопок:
```
1. Создайте Canvas: Right-click → UI → Canvas
2. Добавьте кнопки: Right-click на Canvas → UI → Button
3. Создайте 3 кнопки: Left, Right, Jump
4. Добавьте компонент MobileControls на пустой объект
5. Настройте ссылки на кнопки
```

## ⚠️ Частые проблемы

### Игрок проваливается сквозь пол
- Проверьте что у пола есть Collider2D
- Убедитесь что слои настроены правильно

### Игрок не прыгает
- Проверьте Ground Check Point
- Убедитесь что слой земли назначен в Ground Layer

### Ошибки компиляции
- Убедитесь что все скрипты скопированы
- Проверьте что TextMeshPro установлен (Window → Package Manager)
